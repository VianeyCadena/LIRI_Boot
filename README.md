   # This is LIRI bot

**Author**: Vianey Cadena

LIRI is a simple bot like iPhone's SIRI, in tech words is a command line Node app that takes in parameters and returns data from the following APIs: Bandsintown, Spotify and OMDb.

## How it works

Watch the [video tutorial](https://drive.google.com/file/d/122Rm9jkXQQa6Tm4lstw3JZmQdOnsykdN/view?usp=sharing)

Liri recognice this command line parameters:

* concert-this
* spotify-this-song
* movie-this
* do-what-it-says

To see the magic of LIRI in console just type:

```
node liri concert-this <artist/band>
```
```
node liri spotify-this-song <song name>
```
```
node liri movie-this <movie name>
```
```
node liri do-what-it-says
```
